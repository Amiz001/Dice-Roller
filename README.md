**Dice Roller** is a fun and interactive game created using **HTML, CSS, and JavaScript**. The user selects a guess, rolls the dice, and if the dice matches the guess, the score increments. The game features a **nice UI design** and **sound effects** to make it more engaging.  

> This is one of my early fun coding projects, built purely with frontend technologies and deployed on Netlify.

## Demo

Play the game here: [diceroller365.netlify.app](https://diceroller365.netlify.app)

## Features

- Select a number and roll the dice  
- Score increases when the guess is correct  
- Interactive UI with sound effects  
- Fully frontend-based, no backend required  

## Tech Stack

- HTML  
- CSS  
- JavaScript
